# Website Traffic & User Behavior Analysis

## Overview
This project analyzes website traffic and user behavior using Google Analytics-style data and Excel. The goal is to identify traffic sources with the highest conversion potential and recommend optimization strategies based on user engagement.

## Tools Used
- Excel
- Google Analytics–style dataset

## Key Metrics Analyzed
- Traffic Sources
- Sessions
- Bounce Rate
- Average Session Duration
- Conversions
- Conversion Rate

## Analysis Approach
- Created pivot tables to summarize traffic and user behavior by channel
- Compared engagement metrics such as bounce rate and session duration
- Evaluated conversion performance across acquisition channels
- Interpreted results to identify high-intent and high-volume traffic sources

## Key Insights
- Organic Search and Social channels generate the highest total conversions due to strong traffic volume.
- Referral and Direct traffic show deeper engagement through longer session duration and lower bounce rates.
- Email traffic, while lower in volume, indicates high user intent and potential for scalable conversions.

## Outcome & Recommendations
- Optimize Organic Search landing pages to improve conversion efficiency.
- Scale high-intent channels such as Email and Direct traffic.
- Refine Social traffic targeting to attract more qualified users.
- Maintain and monitor Paid Search performance through keyword optimization.
